## 替换时关键事实（2026-06-04 案例）

- 中欧鼎利债券 C 满 7 天免赎回费 → **可全额无损转换**
- 5 万元中欧鼎利换成 B 档（招商产业债 C + 广发纯债债券 C）后：
  - 波动率从 21% → 2.5-2.8%
  - 最大回撤从 -21.7% → -1.5%~-1.8%
  - 多出的 1-2% 收益由进攻仓（汇添富中证芯片增强等）补

## 注意事项

1. 5 年波动率按"近 5 年净值序列"年化计算，**不预示未来**
2. 候选清单不构成推荐，请结合用户画像与时点市场状态
3. 招商产业债 C 满 30 天才免赎回费（不是 7 天）—— 切换时如需再换出要算这 23 天的窗口
4. 短债基金年化 2.8-3% 是过去 3 年均值，**未来 1-2 年可能更低**（10 年国债 1.6-1.9% 区间）
