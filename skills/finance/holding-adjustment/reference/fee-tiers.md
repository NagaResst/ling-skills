# A/C 类基金赎回费档位与含费成本公式

## A 类与 C 类的核心差异

| 项目 | A 类 | C 类 |
|------|------|------|
| 申购费 | 1 折后 0.06%-0.15% | 0%（免） |
| 销售服务费 | 0% | 0.3%-0.8%/年 |
| 持有 7 天内 | 1.50% 赎回 | 1.50% 赎回 |
| 持有 30 天档 | 0.30% 或更低 | 多数 0% |
| 持有 ≥ 1-2 年 | 0% 或 0.05% | 0% |
| 适合持有期 | 1 年以上 | 1 年以内 |

## 含费成本公式

**A 类**：
```
含费成本 = 净申购价 × (1 + 申购费率折后)
       ≈ 净值成本 + 净值成本 × 0.06%   (天天基金 1 折后)
```

**C 类**：
```
含费成本 = 净值成本   (申购费为 0)
```

## 盈亏平衡线

**用户目标："不赔不赚"清仓**：

```
盈亏平衡卖出净值 = 含费成本 / (1 - 当前档赎回费率)
```

## 止盈/止损触发价

**锁定 Y% 利润**：
```
卖出净值 ≥ 含费成本 × (1 + Y) / (1 - 当前档赎回费率)
```

**允许亏 X%**：
```
卖出净值 ≤ 含费成本 × (1 - X) / (1 - 当前档赎回费率)
```

**注**：分母里的赎回费率是**未来卖出时**所在档位的费率，不是申购时的费率。

## 常见 A 类档位（30 天内高频档）

| 持有天数 | 天弘中证央企红利 50A | 国联安科技动力 | 财通集成电路 A | 汇添富中证芯片 A |
|--------:|----------------------:|---------------:|----------------:|------------------:|
| < 7 | 1.50% | 1.50% | 1.50% | 1.50% |
| 7-30 | 0.30% | 0.75% | 0.50% | 0.30% |
| 30-365 | 0.05% | 0.50% | 0.25% | 0.10% |
| 1-2 年 | 0.05% | 0.25% | 0.10% | 0.05% |
| ≥ 2 年 | 0.05% | 0% | 0% | 0% |

**注**：以上档位以 2026 年 6 月初公开费率为准，逐只基金需以最新招募说明书为准。

## 常见 C 类档位

| 持有天数 | 中欧鼎利债券 C | 财通集成电路 C | 财通成长优选 C |
|--------:|---------------:|----------------:|----------------:|
| < 7 | 1.50% | 1.50% | 1.50% |
| ≥ 7 | 0% | 0% | 0% |

C 类设计意图：免赎回费 + 持续 0.3-0.8% 销售服务费，**适合 7 天 - 1 年的中期持有**。

## 实操示例（2026-06-04 案例）

天弘中证央企红利 50 指数 A：
- 购入 5/27，持有 8 天（7-30 天档，0.30% 赎回费）
- 成本 1.1900，申购费 1 折后 0.06% → 含费成本 1.1906
- 当前净值 1.20
- 盈亏平衡 = 1.1906 / 0.9970 = **1.1942**
- 立刻清仓盈亏：(1.20 × 0.9970 - 1.1906) × 8559.64 = **+49.65 元**
- 再等 22 天到 6/26（30 天-1 年档 0.05%）：盈亏平衡 = 1.1912，假设净值不变 = **+75.32 元**

中欧鼎利债券 C：
- 购入 5/6，到 6/4 已持 29 天（≥ 7 天档，0% 赎回费）
- **满 7 天免赎回费 → 现在可全额无损转换**

## 风险提示

- 档位以"自然日"计，不是"交易日"
- 基金公司保留调整费率的权利，以最新公告为准
- ETF 联接 C 类几乎全部 "满 7 天免赎回费"，是调仓的友好工具
- 持有期跨越档位临界点（如 7 天、30 天）后再操作可省赎回费
