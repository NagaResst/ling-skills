# 净值周期识别算法

## 🎯 目标

从历史净值时序中自动识别完整的涨跌周期，提取统计规律，定位当前所处阶段。

---

## 一、核心概念定义

| 概念 | 定义 |
|------|------|
| **局部顶点（Peak）** | 前后N周内净值最高点，且高于前一底点至少 15% |
| **局部底点（Trough）** | 前后N周内净值最低点，且低于前一顶点至少 15% |
| **一个完整周期** | 底点 → 顶点 → 底点（上涨段 + 下跌段） |
| **上涨周期时长** | 底点日期到顶点日期的月数 |
| **下跌周期时长** | 顶点日期到底点日期的月数 |
| **振幅** | (顶点净值 - 底点净值) / 底点净值× 100% |
| **最大回撤** | 顶点后到底点的最大净值缩水幅度 |

**N 的取值**：
- 数据量 < 100周：N = 4（窗口4周）
- 数据量 ≥ 100周：N = 8（窗口8周，减少噪声误判）

## 二、顶底识别算法（伪代码）

```python
def find_peaks_and_troughs(nav_series, window=8, min_change=0.15):
    """
    识别净值序列中的局部顶点和底点
    
    Args:
        nav_series: list of (date, nav) tuples，按日期升序
        window: 识别窗口（周），前后各 window 周内为局部极值
        min_change: 顶底之间最小净值变化幅度（15%过滤噪声）
    
    Returns:
        peaks: list of (date, nav) — 局部顶点
        troughs: list of (date, nav) — 局部底点
    """
    peaks, troughs = [], []
    values = [nav for _, nav in nav_series]
    
    for i in range(window, len(values) - window):
        window_slice = values[i-window : i+window+1]
        if values[i] == max(window_slice):
            peaks.append(nav_series[i])
        elif values[i] == min(window_slice):
            troughs.append(nav_series[i])
    
    # 过滤：顶底之间净值变化不足 min_change 的视为噪声
    filtered = filter_by_min_change(peaks, troughs, min_change)
    return filtered
```

## 四、当前周期阶段定位方法

### Step 1：确认处于上涨段还是下跌段

- 当前净值 > 上一个确认底点：处于上涨段
- 当前净值 < 上一个确认顶点：处于下跌段（若已跌 > 10%）

### Step 2：判断阶段（上涨段细分）

| 阶段 | 判断条件 | 特征 |
|------|---------|------|
| **A. 底部筑底** | 距底点 < 2个月，净值波动 < 5% | 成交量低迷，市场悲观 |
| **B. 上涨初期** | 距底点 2–6个月，净值涨幅 < 30% | ETF开始流入，情绪回暖 |
| **C. 上涨加速** | 净值涨幅 30%–70%，斜率加大 | ETF大量涌入，媒体关注度上升 |
| **D. 高位震荡** | 净值距历史顶点 < 20%，斜率变缓 | 分歧加大，换手率极高 |
| **E. 顶部确认** | 从高点回落 > 10%，趋势反转 | ETF净赎回，情绪由乐转悲 |

### Step 3：与历史平均时长对比

```markdown
**时长比较**：
- 当前上涨段已持续：15 个月
- 历史平均上涨时长：19 个月（仅2个周期参考，置信度中）
- 剩余预计时长：约 4 个月（若参照历史均值）
- ⚠️ 提示：C1为超级牛市（29个月），C2仅9个月，差异极大，均值参考价值有限。
```

## 五、驱动类型分类

识别到周期后，需结合政策事件库（policy_events.json）判断该周期的主要驱动类型：

| 驱动类型 | 判断依据 | 特点 |
|---------|---------|------|
| **政策驱动** | 上涨起点前后2个月内有国家级重大政策发布 | 持续时间长，回撤也深 |
| **景气驱动** | 行业PMI/出货量数据持续改善 | 上涨稳健，持续时间中等 |
| **情绪反弹** | 无明显基本面改善，从超跌位置反弹 | 上涨时间短（< 6个月），振幅有限 |
| **资金驱动** | ETF大规模净申购，北向持续流入 | 起涨急，但可持续性需基本面配合 |
| **混合驱动** | 以上两种及以上同时存在 | 最强驱动类型，持续时间最长 |
