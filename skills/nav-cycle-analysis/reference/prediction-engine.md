# 预测合成引擎

## 🎯 目标

将「周期位置信号」、「政策事件信号」、「市场活动信号」三个维度的输出，合成为一个具有明确方向、区间和置信度的中期预测结论。

---

## 一、三维信号体系

### 维度 1：周期位置信号（来自 cycle-detection.md）

| 当前阶段 | 信号值 | 说明 |
|---------|-------|------|
| A. 底部筑底 | +2（强多） | 底部买入机会 |
| B. 上涨初期 | +1（偏多） | 趋势刚启动 |
| C. 上涨加速 | 0（中性）| 机会在中间段，现在跟进风险上升 |
| D. 高位震荡 | -1（偏空） | 顶部区域，警惕回撤 |
| E. 下跌调整 | -2（强空） | 调整尚未结束 |

**时长修正**：若当前上涨时长 > 历史平均上涨时长，信号值额外 -1（过期风险）

---

### 维度 2：政策事件信号（来自 policy-event-mapping.md）

**计算方法**：
1. 统计过去4周内已发生的重大政策事件及其方向
2. 统计未来4周内已知政策日历中利好/利空事件数量
3. 综合打分

```
政策信号值 = Σ (已发生事件影响 × 衰减系数) + Σ (预期事件潜在影响 × 确定性系数)

衰减系数：发生后第1周=1.0，第2周=0.7，第3周=0.4，第4周=0.2
确定性系数：已确认=0.8，高概率=0.5，待确认=0.2
```

**量化区间**：-2 到 +2，步长0.5

---

### 维度 3：市场活动信号（来自 SKILL.md 阶段四）

| 指标 | 多头信号 | 空头信号 | 权重 |
|------|---------|---------|------|
| ETF净申购（本周） | > 近6月均值 | < 近6月均值50% | 30% |
| 北向资金净流向（本周） | 连续3日净流入 | 连续3日净流出 | 25% |
| 板块相对强弱 | 连续2周强于基准 | 连续2周弱于基准 | 25% |
| 融资余额变化 | 增加且 < 历史高位80% | 减少或 > 历史高位90% | 20% |

**信号值**：各指标单独计算（-1到+1），加权求和，结果区间-1到+1，再乘以2得-2到+2

---

## 二、综合信号合成

```
综合信号 = 周期信号 × 0.40 + 政策信号 × 0.35 + 市场信号 × 0.25
```

**权重说明**：
- 周期信号权重最高（40%）：净值节律是最基础的约束
- 政策信号次之（35%）：国内政策驱动型市场，政策是关键催化剂
- 市场信号最低（25%）：领先但噪声多，作为辅助验证

---

## 三、信号值 → 预测结论映射

| 综合信号值 | 方向 | 置信度 | 参考振幅（4–12周） |
|----------|------|-------|-----------------|
| +1.5 以上 | ⬆️ 强烈看多 | 75%–85% | +15%~+30% |
| +0.8 至 +1.5 | ⬆️ 偏多 | 60%–75% | +8%~+15% |
| +0.3 至 +0.8 | ↗️ 温和偏多 | 55%–65% | +3%~+8% |
| -0.3 至 +0.3 | ↔️ 中性震荡 | 50%–60% | -5%~+5% |
| -0.8 至 -0.3 | ↘️ 温和偏空 | 55%–65% | -3%~-8% |
| -1.5 至 -0.8 | ⬇️ 偏空 | 60%–75% | -8%~-15% |
| -1.5 以下 | ⬇️ 强烈看空 | 75%–85% | -15%~-30% |

**置信度降级规则**：
- 历史周期数 < 3：置信度 ×0.8
- 数据量 < 100交易周：置信度 ×0.7
- 三维信号方向不一致（多空混合）：置信度 ×0.85

---

## 四、净值区间计算

```
基准预测净值 = 当前净值 × (1 + 参考振幅中值)
上轨净值 = 当前净值 × (1 + 参考振幅上限)
下轨净值 = 当前净值 × (1 + 参考振幅下限)

注意：如当前净值处于历史高位（分位 > 75%），
上轨须额外检查是否超过历史顶点，若超过须注明「突破历史高位，额外风险」
```

## 五、情景分析输出模板

每份预测必须包含三个情景：

```markdown
## 情景分析

### 基准情景（概率：55%）
- **假设**：政策面维稳，市场无重大黑天鹅，行业景气小幅改善
- **净值预测（12周）**：3.40–3.65
- **主要支撑因素**：周期位置B阶段初期；ETF近期持续净流入
- **主要风险因素**：当前上涨时长已超历史均值

### 乐观情景（概率：30%）
- **触发条件**：国家级新能源政策超预期 OR 新能源车月销量创新高
- **净值预测（12周）**：3.65–4.20
- **额外说明**：若触发，需检查是否进入阶段D（高位震荡）

### 悲观情景（概率：15%）
- **触发条件**：监管收紧 OR 锂价持续下跌超10% OR 关税升级
- **净值预测（12周）**：2.80–3.10
- **止损参考**：若净值跌破3.00（当前净值-10.2%），建议重新评估仓位
```

## 七、预测模型自我校正

每次执行本 Skill 时，若距上次预测已过 4 周以上：

1. 对照上次「中期预测」的 M1 预测方向与实际净值走向
2. 若连续2次预测方向错误（基准情景） → 重新评估周期识别的窗口参数（cycle-detection.md 中的 N 值）
3. 若政策信号方向对了但幅度偏差 > 50% → 重新校正 policy-event-mapping.md 中的标准影响参考值
4. 校正记录写入 `净值周期分析/{基金代码}/model_calibration.md`
