## 5 段产业链结构

```
A 上游_芯片本体（壁垒最高，毛利 50%+）
  ├ 算力 CPU: 海光信息、芯原股份
  ├ AI 加速 GPU: 寒武纪、摩尔线程、芯原
  ├ 内存/存储: 兆易创新、佰维存储
  ├ 半导体设备: 中微公司、北方华创、拓荆科技、中科飞测、芯源微
  ├ 半导体材料: 安集科技、雅克科技
  ├ EDA/IP: 芯原股份
  ├ 晶圆代工: 中芯国际
  └ 交换/网络芯片: 澜起科技、盛科通信

B 上游_PCB/基材/封装（毛利 25-40%，放量最快）
  ├ PCB: 沪电股份、生益电子、深南电路、胜宏
  ├ 覆铜板: 生益科技、南亚新材
  ├ 封装设备: 长川科技、华峰测控
  └ 精密加工: 鼎泰高科、大族数控

C 中游_整机服务器（毛利 8-15%，规模最大）
  ├ AI 服务器整机: 浪潮信息、紫光股份、中科曙光
  ├ 白盒/超算: 工业富联（GB200/H200 代工）
  └ IDC: 润泽科技、奥飞数据、世纪互联

D 中游_光模块/CPO（毛利 30-50%，2024-2026 核心）
  ├ 光模块: 中际旭创、新易盛
  ├ 光芯片: 源杰科技、长光华芯
  ├ 光器件: 天孚通信、光库科技
  ├ 光纤光缆: 长飞光纤、永鼎股份
  └ CPO 封装: 罗博特科

E 中游_网络/IDC/电力（毛利 15-25%）
  ├ 通信设备: 烽火通信、亨通光电、中天科技
  ├ 数据中心: 润泽科技、奥飞数据
  ├ 液冷: 英维克、高澜股份、维谛技术
  └ 电源/储能: 科华数据、麦格米特
```

## 各段估值水位（2026-06 估算）

| 段 | PE 区间 | 投资属性 |
|----|--------|---------|
| A 芯片本体 | 50-200x | 弹性最大，国产替代 |
| B PCB/基材 | 25-40x | 业绩兑现期 |
| C 整机服务器 | 20-30x | 低估值确定 |
| D 光模块/CPO | 40-80x | 主升浪，估值最贵 |
| E 网络/IDC | 15-25x | 低估值补涨 |

## 政策传导链

```
政策 → 资本开支 → A 设备/材料 → A 芯片 → B PCB/封装 → C 整机 → D 光模块/CPO → E IDC/电力
                                                                  ↓
                                                                应用/算力券
```

**"政策在建设 AI 算力"是 A→E 全链机会**，但**纯度与弹性不同**：
- 弹性最大：A、D（最贵、波动最大）
- 业绩最确定：B（财报兑现期）
- 估值最便宜：C、E（"卖铲子"但毛利薄）
